# Build language
How build `Yummy`
```bash
cmake --preset=release # or debug/dev
cmake --build --preset=release
```

---

How get **preset list**
```bash
cmake --list-presets
```

---

В проекте используюется **vcpkg**. Настоятельно рекомендовано установить `vcpkg` с официального репозитория по инструкции.
Сборка может потребовать некоторого времени.
